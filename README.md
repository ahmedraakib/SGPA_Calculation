# SGPA Calculator

This program calculates your Semester Grade Point Average (SGPA) based on the grades you receive in your courses. It uses the grading system provided to convert your grades to GPA and then calculates your SGPA.

## How It Works

1. Run the program.
2. Enter the number of courses you're taking this semester.
3. For each course, enter the grade you received (choose from A/B/B+/B-/C/C+/C/D+/D/F) and the credit hours of the course.
4. The program will calculate and display your SGPA for the semester.

## Grading System

The program uses the following grading system:

- A: 4.0
- B+: 3.7
- B: 3.4
- B-: 3.1
- C+: 2.8
- C: 2.5
- C-: 2.2
- D+: 1.5
- D: 1.0
- F: 0.0
